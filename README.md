🧰 WindowsCostumizeWizard

🇬🇧 English
📌 Description

WindowsCostumizeWizard is a graphical tool for working with DISM commands in Windows.
It allows system operations to be executed through a GUI instead of manually typing commands in the terminal.

The application works with:

Features (Windows optional components)
Packages (updates and system packages)
Appx (UWP applications)

⚙️ Features

GUI-based DISM command execution
Management of Features / Packages / Appx
Offline and online mode support
Lightweight OS customization tool

🧩 Dependencies

External tools:

oscdimg and wimlib (user choice)

Libraries:
Microsoft.WindowsAPICodePack.dll
Microsoft.WindowsAPICodePack.Shell.dll
Newtonsoft.Json.dll
